# day-number
*Day of the Year Calculator*  C# console app calculating day of year for given date (MM/dd/yyyy).  Calculates day of year, checks leap years using arrays &amp; loops.  Uses: Date Parsing, Arrays, Loops, Methods, Error Handling  Examples: "02/28/2020" -> "59 days into 2020", "02/29/2020" -> "2020 is a leap year"
